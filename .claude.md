# FCIT SRS — Project Journey
*Claude Sonnet 4.6 · UniSmart Academic Platform · May 2026*

## What This Is
Full-stack university Student Registration System for FCIT, Tanta University.
Three actors: Admin · Doctor · Student — Arabic RTL UI, bylaw-enforced registration.

**Stack:** Node.js · Express · PostgreSQL 16 · React (single-file) · Prometheus v3.4.0 · Grafana 11.6.0 · Docker Compose

## Phase 3 — Registration Bug Fix (Critical)
**Root cause:** `bylaw.service.js` checked `add_drop_deadline` even during `registration` status — Spring 2026 deadline was March 2026, so ALL courses returned blocked.
**Fixes:** Status-aware deadline check, updated semester dates, `PATCH /admin/semesters/:id/dates`, `POST .../toggle-registration`, rebuilt AdminSemestersPage.

**PostgreSQL startup packet fix:** Prometheus was scraping `postgres:5432` directly with HTTP.
**Fix:** Added `postgres_exporter:v0.17.1` sidecar, updated prometheus.yml to scrape `:9187`, updated images: Prometheus v2.51→v3.4.0, Grafana 10.4→11.6.0.

**Prerequisites:** 72 relationships in `migration_prerequisites.sql` covering full FCIT CS/IS/IT/SE curriculum.
